Build My Blog

安装

Hexo架构的博客基于node。

node.js

可从 https://nodejs.org/zh-cn/ 中下载长期支持版，并通过npm指令安装hexo。国内的网络可以通过淘宝来下载npm的镜像，方便之后的操作不受网络问题影响。
npm install -g cnpm –registry=https://registry.npm.taobao.org

hexo

通过cnpm安装hexo的指令为
cnpm install -g hexo-cli

在本地建立博客

在本地建立空文件夹

例 在地址Users/chaohu/处，通过mkdir blog，建立空的文件夹来作为博客本地的存储地址

在本地生成hexo博客

进入到新建的文件夹后，输入指令
sudo hexo init
此时博客框架已经搭好
要建立新文章，输入指令
hexo new “file name”
遍会在blog文件夹中的../source/_posts/ 中生成文档。文档内容可以通过vscode编辑。
之后，在清理过之后（hexo clean），输入hexo g指令便可以产生新的博文

将本地的博客通过github推送至网络

建立库

在github的账号中建立库，库的名字必须与用户名一致

推送部署插件

第一次操作需在本地安装，可输入指令
cnpm install –save hexo-deployer-git
并在_config.yml文件中的末尾处设置type：git; repo: 库的地址; branch: master
执行推送部署指令
hexo d
之后输入github账号用户名密码
